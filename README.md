В результате расширения функционала парсера можно было бы создать более совершенную систему рекомендаций фильмов,
которая будет использовать данные из различных социальных сетей, посвящённых кинематографу.

Задача была сформулирована следующим образом: разработать парсер,
который принимает уникальный идентификатор пользователя на платформе кинопоиск и возвращает информацию о его оценках фильмов.
